Welcome To Venice

Project Venice was started with the question of how well could Genetic Algorithms or rather Genetic Programming be applied to generating successful buy/sell rules on the stock market? With some promising preliminary results, the project scope expanded. The current goals are: to provide stock market analysis, portfolio management and to provide a user friendly interface. With these new goals, the project was re-written from scratch and is now working towards re-implementing the Genetic Programming.
