Contribute

You can contribute to Project Venice in several different ways:

Develop

Venice is an Open Source project, the source code is distributed under the GNU General Public License (GPL). This gives developers the ability to view and modify the source code, while being safe in the knowledge that their contributions are protected under the GPL. Code contributions are always welcome. If you plan to develop, or already developing for Venice please let us know.

For suggestions about what to add, here is a list of the planned future features

Donate

All donations to Project Venice will be greatly appreciated. You can donate books,money, or even just a postcard.

Suggest

Perhaps you tested Venice and found it did not have all the features you require. Perhaps the user interface doesn't work how you think it should. Perhaps it didn't work at all! If you don't let us know, then we might not know to fix it.

Translate

The latest version of Venice supports internationalisation and can display languages other than English. Translating the user interface text is easy. If you are interested in translating, please contact us. Currently, Venice has been translated into Chinese, French, Italian, Polish, Swedish and Catalan.
