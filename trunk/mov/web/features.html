Feature List

The current version of Venice supports the following features:
  • Analysis
    • Automated Paper Trade on buy/sell rules
    • Automated Paper Trade on family of buy/sell rules
  • Charting
    • Chart Advance/Decline
    • Chart Bollinger Bands
    • Chart High-Low
    • Chart Momentum
    • Chart Moving Average
    • Chart On Balance Volume (OBV)
    • Chart Open/High/Low/Close/Volume
    • Chart Standard Deviation
  • Gondola - language for analysing quotes
  • List day's quotes in Table
    • Sort by Most Active
    • Filter by rule
    • Apply functions to quotes
  • Portfolio Management
    • Supports multiple portfolios
    • Portfolios support multiple cash/stock accounts
    • Chart Portfolio value over time
  • Quotes
    • Access quotes via files or MySQL
    • Import quotes from EzyChart, InsightTrader or MetaStock formats
  • Online Help
