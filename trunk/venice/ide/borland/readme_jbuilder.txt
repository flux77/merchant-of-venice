This file contains some information about runing venice in 
JBuilder 2005.

Please set the Ant home property in menu 'Project - 
Project Properties', 'Build - Ant' to the path of your
ant installation. Recommended is ant version 1.7.1.

Make sure you use jdk_1.4.2_19. The jdk_1.4.2_05 shipped
with JBuilder 2005 will not work.
